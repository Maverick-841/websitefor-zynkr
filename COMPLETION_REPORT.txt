🎓 WHAT YOU LEARNED

CORS Preflight:
  - Browser requires matching credentials config on both client and server
  - Client needs: credentials: 'include' in fetch
  - Server needs: credentials: true in CORS
  - Without both, preflight fails and browser blocks actual request

Common Mistake:
  - "route not found" doesn't mean endpoint is missing
  - Usually indicates CORS preflight failure
  - Check browser Network tab for failed OPTIONS request

Testing Importance:
  - Always test with actual files before deployment
  - File upload errors often hide CORS issues
  - Backend logs are your friend (look for [Resume Upload] prefix)
